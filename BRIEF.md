# Konzelhaus Weeknight Dinners — Build Brief

Build a beautiful single-page meal planning website. Use React (via CDN/unpkg) and Tailwind CSS (via CDN). Everything in ONE index.html file — no build tools, no npm.

## Design Direction
- Warm, calm aesthetic — cream/warm white background with ONE accent color (amber or terracotta)
- Premium typography — use Google Fonts (e.g., DM Serif Display for headings, Inter for body)
- Apple-like negative space — don't crowd things
- No flashy animations, no emojis in the UI
- Mobile-first, responsive

## Layout
- Hero section with family-friendly headline ("Konzelhaus Weeknight Dinners")
- Grid of recipe cards (3 columns desktop, 1 column mobile)
- Each card: recipe name, prep time, short description, link to full recipe
- Cards should have subtle hover effect
- Footer with "Made with love for the Konzelmanns"

## Technical
- Single index.html file
- React 18 via unpkg CDN
- Tailwind CSS via CDN
- Google Fonts via CDN
- No npm, no build step, no node_modules
